Backgrounds

Image slider

This version is in the file index.html. If you want to use the image slider you have to edit the file js/custom.js and change the variable image_slider:

...
var image_slider = true;
...

To change the images in the slider you have to edit the file js/image_slider.js

...
{image : 'http://placehold.it/1280x720&text=Image+1', title : 'White - Creative...'},
{image : 'http://placehold.it/1280x720&text=Image+2', title : 'White - Creative...'},
{image : 'http://placehold.it/1280x720&text=Image+3', title : 'White - Creative...'},
...

Single image

This is the version in the file single_image.html. To change the image you have to edit the file js/single_image.js

...
{image : 'http://placehold.it/1280x720&text=Image+1', title : 'White - Creative...'},
...

Single image with gradient

This is the version in the file single_image_gradient.html. To change the image you have to edit the file js/single_image.js

{image : 'http://placehold.it/1280x720&text=Image+1', title : 'White - Creative...'},
...

And if you want to change the gradient colors, you have to change the class #gradient-1 in the file style.css

Gradient version

If you want to change the gradient colors, you have to change the class #gradient-2 in the file style.css

Video versions

There are three options with the video version. Your own video, a video from Youtube (or a list), and a video from Vimeo.
These scripts have a lot of problems in mobile and tablet devices. In this cases, the template changes the video by an image. The class poster-image is added to the div bgimg.
To change the class poster-image, you have to edit the file style.css:

...
.poster-image {
background:url('http://placehold.it/1280x720&text=Image+1');
...

Video in your server

To configurate your own video you have to use the index_video.html version and edit the file js/video.js
If you want the video with sound you have to edit the line 7:

...
BV.show('vids/video.mp4');
...

But, if you don't want sound in the video you have to edit the file in this way:

...
BV.show('vids/video.mp4',{ambient:true});
...

Youtube video

To use a video from Youtube you have to use the file index_video2.html and edit it.
You have to copy the video url from Youtube and type it in the code.

...
data-property="{videoURL:'http://www.youtube.com/watch?v=qkfGP4UTpjE', containment:'#video-content',
autoPlay:true, mute:false, startAt:0, opacity:1, ratio:'4/3', addRaster:true}"
...

Vimeo video

To use a video from Vimeo you have to use the file index_video3.html version and edit it.
You have to copy the video id from the Vimeo website and type it in the code.

...
data-property="{videoURL:'52262947', containment:'#video-content',
align:'bottom,center', showControls:false, autoPlay:true, loop:true, mute:true, startAt:0, addRaster:true}"
...

Config

Menu

Remember to use the classes right-margin and left-margin to create the space between the buttons where the logo is placed.

Contact form

For the contact form, you need to configure some variables located in the file php/config.php:

...
define('TO_NAME','name'); //put your name
define('TO_EMAIL','email@domain.com'); //put your email
...
define('SMTP_HOST','smtp.domain.com'); //for example gmail: smtp.gmail.com
define('SMTP_USERNAME','email@domain.com'); //put your email
define('SMTP_PASSWORD','password'); //put your password
define('SUBJECT','Contact from your website White HTML'); //change if you want the subject
...

Page transitions

There is the option to change the page effect transition. You can view all the options in the next page: http://tympanus.net/Development/PageTransitions
If you want that, you have to change the attribute data-animation in the menu

...
<ul class="nav navbar-nav"> <li><button class="pt-trigger" data-animation="22" ... ...

Map

First of all, you need to get a Google Api Key and paste it in your application, include it as a key parameter value (index.html)

... src="http://maps.google.com/maps/api/js?key=YOUR_API_KEY&callback=initMap" ...

If you want to change the map coordinates, you have to edit the file js/map.js

...
var latitude = 42.814681,
longitude = -1.6495237000000316,
map_zoom = 16;
...
You can get the map coordinates in this website
